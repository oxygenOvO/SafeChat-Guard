# 语义分类实验报告

> **实验日期**：2026-07-15  
> **实验人员**：B组（语义分类与风险分级）  
> **数据版本**：raw_train_v3.csv（V3平衡数据 + violation_sentences 扩充 + 过采样强化）

---

## 1. 实验目的

比较 **词级（word）** 与 **字符级（char）** 特征方案对中文文本风险分类的影响，确定最终采用的方案。

---

## 2. 实验设置

### 2.1 数据规模
- **总样本数**：1860 条
- **类别分布**：
  - violence: 558
  - normal: 378
  - sensitive: 328
  - ad: 300
  - porn: 296

### 2.2 特征提取
- **向量化器**：`TfidfVectorizer`
- **max_features**：10000
- **min_df**：2
- **max_df**：0.8
- **分类器**：逻辑回归（`LogisticRegression`，多分类）

### 2.3 实验方案对比
| 方案 | analyzer | ngram_range |
|------|----------|-------------|
| Word 基线 | word | (1, 2) |
| Char 方案 | char | (2, 5) |

### 2.4 训练/测试拆分
- 训练集：1488 条（80%）
- 测试集：372 条（20%）
- 随机种子固定，保证两组实验数据一致

---

## 3. 实验结果

### 3.1 整体指标对比

| 指标 | Word 基线 | Char 方案 | 提升 |
|------|-----------|-----------|------|
| **测试集准确率** | 94.89% | **98.12%** | **+3.23%** |
| **Macro Precision** | 95.96% | **98.54%** | **+2.58%** |
| **Macro Recall** | 93.90% | **97.73%** | **+3.83%** |
| **Macro F1** | 94.51% | **98.09%** | **+3.58%** |

### 3.2 各类别详细指标（Char 方案）

| 类别 | Precision | Recall | F1-Score | Support |
|------|-----------|--------|----------|---------|
| ad | 1.00 | 1.00 | 1.00 | 60 |
| normal | 0.96 | 1.00 | 0.98 | 75 |
| porn | 1.00 | 0.93 | 0.96 | 59 |
| sensitive | 1.00 | 0.95 | 0.98 | 66 |
| violence | 0.97 | 1.00 | 0.98 | 112 |

### 3.3 混淆矩阵（Char 方案）
[[ 60 0 0 0 0] -> ad
[ 0 75 0 0 0] -> normal
[ 0 0 55 0 4] -> porn (4条误判为violence)
[ 0 3 0 63 0] -> sensitive (3条误判为normal)
[ 0 0 0 0 112]] -> violence


---

## 4. 结论

- **Char 方案在所有核心指标上均优于 Word 方案**，准确率提升 3.23%，Macro F1 提升 3.58%。
- 对中文文本，字符级 N-gram（2-5）能更有效地捕获字形和构词信息，克服中文分词不准确的问题。
- **最终决定**：采用 **Char 方案** 作为语义分类器的特征提取方式。
